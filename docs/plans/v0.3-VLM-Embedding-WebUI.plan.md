## 1. 真实 VLM 接入

**方案：**

- `OpenAICompatibleVLMAnalyzer`：httpx + `/chat/completions` + base64 图
- Pydantic 校验 JSON schema
- `FallbackVLMAnalyzer`：API 失败降级启发式
- 注入前台窗口上下文

**配置：**

```yaml
vlm:
  provider: openai_compatible
  fallback_heuristic: true
```
