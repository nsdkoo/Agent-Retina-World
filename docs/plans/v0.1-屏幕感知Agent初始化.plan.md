## 目标

搭建可本地运行的桌面屏幕感知 Agent **最小全链路**：

```
截图 → pHash 去重 → VLM 理解 → Activity 聚合 → SQLite 记忆 → 日报/待办
```

## 方案

### 模块划分

| 模块 | 路径 | 职责 |
| --- | --- | --- |
| capture | `capture/screen.py` | mss 定时截图 |
| dedup | `dedup/hasher.py` | L1 感知哈希 |
| understand | `understand/vlm.py` | Mock / OpenAI 兼容 VLM |
| activity | `activity/store.py` | 时间线聚合 + SQLite |
| proactive | `proactive/service.py` | 日报、待办 |
| pipeline | `pipeline.py` | 编排 |

### CLI

- `python main.py once` — 单帧
- `python main.py watch` — 定时采集
- `python main.py report` — 日报

### 配置

- `config.example.yaml`：capture / dedup / vlm / memory
- 默认 Mock VLM，无 API 可跑

## 交付物

- `requirements.txt`：mss, Pillow, imagehash, pyyaml, pydantic, httpx
- `docs/architecture.md` + 架构图
