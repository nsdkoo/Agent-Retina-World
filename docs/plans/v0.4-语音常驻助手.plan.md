# Plan（过程重建）：v0.4 语音常驻助手

> **类型**：过程重建  
> **日期**：2026-06-23  
> **用户原话**：呼唤名字操作；不要打字、不要对话窗口；常驻桌面/侧边栏；录音常驻  
> **版本**：v0.4.0

---

## 目标

实现 **无对话窗口的语音操控层**，与屏幕感知 Pipeline 并行：

- 用户说「小光，截图」→ 立刻执行
- 不依赖 Chat UI，靠 **唤醒词 + 规则意图**

## 交互设计

| 要素 | 设计 |
| --- | --- |
| 唤醒词 | Retina / 小光 / 光光（可配置） |
| STT | SpeechRecognition + Google 在线（初版） |
| TTS | pyttsx3 播报结果 |
| UI | 右侧 **侧边栏** 300px，置顶，显示状态与日志 |
| 意图 | 正则规则：截图、打开 URL/App、分析、日报、时间线、统计 |

## 模块

```
voice/
  assistant.py    # 主循环 listen → parse → execute → speak
  intents.py      # IntentType + parse_intent
  executor.py     # CommandExecutor → pipeline / webbrowser / subprocess
  listener.py     # SpeechListener + Speaker
  sidebar.py      # VoiceSidebar UI
```

## CLI

```powershell
python main.py voice
python main.py voice --ui sidebar
python main.py voice --no-ui
```

## 与屏幕感知关系

| 模式 | 命令 | 行为 |
| --- | --- | --- |
| 主动操控 | `voice` | 用户说话 → 即时命令 |
| 后台感知 | `watch` | 静默采集时间线 |

可同时运行两个终端。

## 已知限制（v0.4 结束时）

- 需网络（Google STT）
- 每次需喊唤醒词
- 侧边栏较大，用户后续要求悬浮球 → v0.5
