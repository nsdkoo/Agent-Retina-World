## 目标

在 v0.4 语音助手基础上，完成三项体验升级。

## 1. 免唤醒连续对话

**方案：**

- 唤醒后进入 `session_mode`，默认 60 秒
- 会话内直接说指令，无需再喊名字
- 「退出 / 没事了」结束会话
- 悬浮球 **紫色** = 会话中

**配置：**

```yaml
voice:
  session_mode: true
  session_duration_seconds: 60
```

## 2. 离线语音识别（Vosk）

**方案：**

- `voice/offline_stt.py`：VoskSTT / GoogleSTT / auto
- `auto`：有本地模型则离线，否则 Google
- `--download-model` 下载 vosk-model-small-cn-0.22（~42MB）

**依赖：** vosk, SpeechRecognition, pyttsx3

## 3. 悬浮球 UI（默认）

**方案：**

- `voice/floating_ball.py`：52px 圆球，置顶、可拖
- 颜色：灰 idle / 绿 listening / 蓝 processing / 紫 session
- 双击展开日志，右键菜单
- `voice.ui: ball | sidebar | none`，默认 ball
